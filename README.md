# Time Recorder — Учебный проект

Простой веб-проект на Node.js, Express и SQLite. Позволяет записывать время и дату кликов пользователя, показывать последние 4 записи и блокировать слишком частые клики.

---

## Что делает проект

- Отображает часы в реальном времени.
- Кнопка "Записать время" сохраняет текущую дату и время в базу.
- Таблица показывает последние 4 записи.
- Кнопка "Очистить таблицу" удаляет все записи.
- Защита от спама: больше 5 кликов за 3 секунды → блокировка на 10 секунд.
- При загрузке страницы таблица сразу показывает последние записи.

## Как работает проект

### Бэкенд
- Сервер Node.js + Express.
- SQLite хранит записи с датой и временем.
- Endpoint'ы:
  - `POST /record` — сохраняет дату и время, проверяет спам.
  - `GET /records` — возвращает все записи.
  - `POST /clear` — очищает таблицу.
- Время берётся с сервера (`new Date()`), блокировка по IP.

### Фронтенд
- HTML показывает часы, таблицу и кнопки.
- JS обновляет часы каждую секунду.
- JS отправляет запросы на сервер, обновляет таблицу.
- Если пользователь заблокирован — выводится alert с сообщением.
